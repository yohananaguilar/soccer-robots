# soccer-robots

Repositorio da extensao universitaria de robotica dedicado ao estudo de microcontroladores e ao desenvolvimento de robos de futebol autonomos.

Nesta branch, o foco esta nos materiais didaticos das aulas, com exemplos de firmware e diagramas de placa usados nas atividades praticas.

## Estrutura desta branch

- aulas_scripts/: materiais das aulas organizados por data e tema.
- docs/: materiais de apoio e documentacao complementar.
- README.md: visao geral da branch de aulas.

## Organizacao das aulas

Cada pasta dentro de aulas_scripts contem:

- firmware/: codigo-fonte trabalhado em aula.
- hardware/: diagrama da placa em formato EAGLE.
- README proprio com resumo, conteudo e observacoes da atividade.
