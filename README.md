# soccer-robots
Repositório dedicado ao desenvolvimento de um time de robôs de futebol autônomos (Goleiro, Zagueiro e Atacante) utilizando microcontroladores ESP32 e C++. Projeto desenvolvido na extensão universitária de robótica.

## 📖 Materiais de Estudo
Para aprofundar os conhecimentos em Arduino e C++ aplicados neste projeto, recomendamo a leitura:

- **Livro:** *Arduino in Action* (Martin Evans, Joshua Noble, Jordan Hochenbaum).
- **Idioma:** Inglês (Original).
- [Acesse a versão online aqui](https://livres.ycharbi.fr/Livres/ebook%20informatique/Informatique/Arduino%2C%20Raspberry%20Pi/Arduino/Arduino%20in%20Action%20%282013%29%20-%20%5BManning%5D%20-%20Martin%20Evans%2C%20Joshua%20Noble%2C%20Jordan%20Hochenbaum.pdf)
