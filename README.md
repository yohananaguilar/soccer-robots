# soccer-robots
Repositório dedicado ao desenvolvimento de um time de robôs de futebol autônomos (Goleiro, Zagueiro e Atacante) utilizando microcontroladores ESP32 e C++. Projeto desenvolvido na extensão universitária de robótica.
