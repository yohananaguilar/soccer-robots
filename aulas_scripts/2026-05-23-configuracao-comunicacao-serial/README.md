# Aula 04 - Configuracao da comunicacao serial

Material da aula realizada em 23/05/2026.

## Conteudo

- firmware/configuracao-comunicacao-serial.cpp: exemplo basico de inicializacao da comunicacao serial e envio periodico de mensagens para o monitor serial.
- hardware/configuracao-comunicacao-serial.brd: layout da placa no EAGLE para o circuito utilizado na atividade.

## Observacao

O script destaca a configuracao inicial da porta serial com Serial.begin e o envio continuo de mensagens com Serial.println, servindo como base para compreender taxa de transmissao, monitor serial e depuracao de sistemas embarcados.
