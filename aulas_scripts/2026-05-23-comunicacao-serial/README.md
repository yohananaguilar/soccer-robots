# Aula 05 - Comunicacao serial com controle de LED

Material da aula realizada em 23/05/2026.

## Conteudo

- firmware/comunicacao-serial.cpp: exemplo de leitura de comandos pela porta serial para controlar o estado de um LED conectado ao pino digital 13.
- hardware/comunicacao-serial.brd: layout da placa no EAGLE para o circuito utilizado na atividade.

## Observacao

O script utiliza a leitura de caracteres enviados pelo monitor serial para ligar e desligar o LED, reforcando o fluxo completo de entrada e saida de dados pela comunicacao serial e a integracao entre software e hardware em atividades interativas.
