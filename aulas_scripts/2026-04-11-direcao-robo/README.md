# Aula 03 - Controle de motores com ponte H

Material da aula realizada em 11/04/2026.

## Conteudo

- firmware/movimentacao-robo.cpp: exemplo de controle de motores com ponte H para deslocamento do robo para frente, parada, retrocesso e mudanca de direcao.
- hardware/movimentacao-robo.brd: layout da placa no EAGLE para o circuito utilizado na atividade.

## Observacao

O script apresenta a configuracao dos pinos de controle da ponte H e organiza os movimentos do robo em funcoes separadas, reforcando o uso de saidas digitais, acionamento de motores e estrutura modular para facilitar testes e ajustes de movimentacao.
