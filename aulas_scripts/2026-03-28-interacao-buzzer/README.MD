# Aula 02 - Interacao com buzzer

Material da aula realizada em 28/03/2026.

## Conteudo

- firmware/interacao_buzzer.cpp: exemplo de controle de buzzer com a funcao tone para reproduzir uma sequencia de notas.
- hardware/interacao_buzzer.brd: layout da placa no EAGLE para o circuito utilizado na atividade.

## Observacao

O script utiliza o pino digital 13 para acionar o buzzer e organiza a melodia em blocos sequenciais dentro da funcao loop, reforcando o uso de saida digital, temporizacao e estruturas basicas de programacao vistas na aula.
