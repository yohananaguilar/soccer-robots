# Aula 01 - Introducao ao ESP32

Material da aula realizada em 21/03/2026.

## Conteudo

- firmware/blink_led.cpp: exemplo basico de acionamento de LED com alternancia entre nivel logico alto e baixo em intervalos de 1 segundo.
- hardware/blink_led_arduino_uno_shield.brd: layout da placa no EAGLE para o circuito utilizado na atividade.

## Observacao

O script utiliza o pino digital 13 como saida para controle do LED, reforcando os conceitos iniciais de configuracao com pinMode, escrita digital com digitalWrite e temporizacao com delay, que serviram como base para a introducao a programacao embarcada na aula.
